jQuery Tools Validation is something that hasn't existed before: a compact, standards based and easy HTML form validation tool. It relies on the new HTML5 input data types and takes the best parts of Web Forms 2.0 standard. Combined with the power of jQuery we can transform the fairly complex Web Forms standard into pure amazing.

Use HTML5 input types now! HTML5 provides new input types such as number, email and url. Or you can specify additional conditions such as max, min, required or pattern. These work out of the box on every major browser. You can also use amazingly flexible API to add your own input types or conditions. You can take full advantage of the jQuery selector engine.
