Essential tools for modern websites

Let's face it: do you really need drag-and-drop, resizable windows or sortable lists in your web applications? Websites are not desktop applications. They are different.

What you really need are tabs, tooltips, accordions, overlays, high usability, striking visual effects and all those "web 2.0" goodies that you have seen on your favourite websites.

This library contains the most useful JavaScript tools available for today's website. All of these tools can be used together, extended, configured and styled. You'll have many different widgets and new personal ways of using the library.

This library is open source and dual licensed under MIT and GPL 2+ licenses.

Small size. Free CDN. High performance.

This library weighs ${v.jqtSize}Kb. This is possible with good programming habits and compressed files

The file size is not the primary reason for high performance. The ability to combine tools into single file makes it performance friendly. According to Yahoo experts, this is the first rule for making highly responsive websites.

Yahoo's second rule state: "Use a Content Delivery Network". Well, we will give you one for free! It has been configured with the proper expire headers and compression settings making the file load with the maximum performance possible. You can event use the CDN in production environments.


For beginners

You don't need to be a programmer to use these tools. We have lots of demos and you can copy their source code to your site and things will work.

Our demos use minimal HTML, CSS and JavaScript syntax and the crucial parts have been documented. You can use them as a template for your own work.

And if you find suitable graphics you can freely use them on your site. In fact we have downloadable zip files for graphics designers that can be used as a starting point.

Each tool is used in a similar manner and by learning one tool you can easily learn the other tools.

For serious programmers

Each tool does only what it is supposed to do and nothing else. If you want modify this default behaviour, we offer clever ways of altering and/or extending it.

The API design is probably the strongest area of these tools. The design patterns are understandable. We follow the UNIX tradition of providing tools - not policy.

CSS is never mixed with code. This library takes this seriously. CSS is the ultimate tool for styling - not JavaScript.

And above all, the library is constantly maintained and updated. Expect to see new releases coming out on a regular basis. The source code is hosted on Github. Enjoy!
