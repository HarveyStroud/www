This demo introduces you to some of the configuration possibilities of the dateinput. Following dateinputs are initialized with single $(":date").dateinput() call with a shared configuration. Additionally each input uses some HTML5 attributes for input specific configurations.

HTML5 attributes

Native HTML5 date input supports two attributes min and max that can be used to set boundaries for the selectable day. The value must be in RFC 3339 format such as: 2010-12-22 but this tool also accepts a numeric value meaning a number of days from the present day. data-value is the primary means of providing a default value for the input.
