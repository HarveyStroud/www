The image below opens up an overlay that contains a huge image. It is not loaded from the server until the overlay is opened making the initial page load faster.

<%@ include file="overlay.htm" %>

The JavaScript configuration for this setup is simple.

$("#trigger").overlay({lazyload: false});

HTML coding

Both background images and regular images are lazyloaded when the lazyload property is enabled. On this overlay we have a normal image to be loaded when the overlay is opened.
