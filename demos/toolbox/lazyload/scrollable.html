On following scrollable the images and background images are not loaded until they become visible. The installation is as simple as enabling the lazyload variable as follows.

$(".lazyscroll").scrollable({ size: 1, lazyload: true }); <%@ include file="scrollable.htm" %>

HTML coding

Both background images and regular images are lazyloaded when the lazyload property is enabled. On our scrollable we have background images on the scrollable items and the last item has a small "normal" image which is lazyloaded.

CSS coding

We won't go into details how a scrollable is laid out. For that you should start with the minimal setup for scrollable. Here we will only focus on the issues that relate to lazyloading. In our example we have made a minimal styling for the "progress indicator" which simply says "Loading …".

.progress { margin-left:15px; position:absolute; top:15px; font-size:11px; color:#fff; }
