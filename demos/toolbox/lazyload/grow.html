Lazyload has a built-in grow effect which causes the wrapper element to grow once the element has been loaded. The dimensions are either image or the loaded DIV dimensions. This feature allows you to build interesting features to your JavaScript applications. Think of lazyloaded and growing slideshows, tooltips or overlays.
