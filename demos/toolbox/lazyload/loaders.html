Lazyload is capable of loading images, background images and HTML elements. Here we have an example of all of these types. Click on the "load assets" button.

<%@ include file="loaders.htm" %>

HTML coding

Images and background images are defined normally. Background images can also be defined via an external stylesheet and they will still be lazyloaded. If you want to lazyload HTML the syntax can be taken from the third element. It is a wrapper element (typically div) enclosing a single anchor tag where the page to load is specified in the href attribute. This is standard HTML and is viewable by search engines and JavaScript disabled browsers too.

${html}
