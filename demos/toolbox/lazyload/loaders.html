Lazyload is capable of loading images, background images and HTML elements. Here we have an example of all of these types. Click on the "load assets" button.

<%@ include file="loaders.htm" %>

HTML coding

Images and background images are defined normally. Background images could have been defined via external stylesheet also and they will be lazyloaded too. If you want to lazyload HTML the syntax can be seen from the third element. It is a wrapper element (typically div) enclosed with a single anchor tag where the loaded page is specified in href attribute. This is standard HTML and is viewable by search engines and JavaScript disabled browsers too.

${html}
