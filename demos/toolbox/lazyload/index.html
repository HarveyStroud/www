This page has a couple of heavy images referenced from Flickr. Note that this page loaded quickly. The images are below the documentation section and they will only be loaded once they become visible. No redundant loading is done. Scroll down the page and you'll see lazyloading in action.

HTML coding

We have enclosed the images inside a DIV so that we can select them easily with jQuery.

JavaScript coding

Here we initialize lazyloading for the images. We supply true to the constructor which is identical to lazyload({loadOnScroll: true}); meaning that we load images based on the scrolling position. There are other means of loading also that you will see on later demos.

$("#loadables img").lazyload(true);
