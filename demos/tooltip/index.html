Here we have four IMG tags and their title attribute work as the tooltip. This kind of simple syntax will also show browser's standard tooltips even thought JavaScript is not enabled. This whole setup can be enabled with one JavaScript one-liner.

HTML coding

You need two things: the elements that show the tooltips when the cursor is placed on top of them, these elements are called triggers. Then you need one or more tooltip elements. In this setup we have single tooltip element that works for all of the triggers:

 

Trigger elements can be anything such as A, DIV, SAMP and TABLE elements.

CSS coding

The tooltip should be initially hidden by the CSS by setting display: none. We have no specific styling for the trigger elements.

JavaScript coding

Tooltip activation starts by selecting the trigger elements with jQuery. Here we select all img tags that have a title attribute. We supply one argument for the tooltip initialization call which is a jQuery selector to the tooltip element.

${js}

The default behaviour is that the tooltip is positioned on the top/center of the trigger and it slides upwards. Of course, the positioning and the sliding effect can be altered in the configuration as seen in the upcoming demos.

Enclosing your call inside $(document).ready performs the script right after the document is scriptable. Read more about that in the User's Manual.
