HTML coding

Here we have a form that has valid HTML5 syntax. We have used email, url and number inputs as well as normal text input with pattern attribute. requred attribute is used to indicate mandatory field. maxlength attribute makes it impossible to enter more than 30 characters.

<%@ include file="myform.jsf" %>

JavaScript coding

This is the simple part. Simply initialize validator and you'll get all HTML5 niceties.

$("#myform").validator();

You need to place the script block after the HTML code or you can alternatively use jQuery's $(document).ready to execute the script as soon as the webpage's Document Object Model (DOM) has been loaded. Read more about that from the User's Manual.

CSS coding

We have given some spice for the error message. Note that height property should be give for the error message to ensure consistent positioning for every browser. We have defined label element as a block level element so that it appears on it's own "line".
